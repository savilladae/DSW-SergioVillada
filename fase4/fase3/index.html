Propósito del sitio

Este sitio esta orientado a generar conocimientos en el ámbito del Diseño de sitios Web y como parte puntual en esta modulo con enfoque a las temáticas relacionadas con: Etiquetas Básicas de HTML, atributos y valores.

Introducción

En este sitio web encontraremos contenido temático y didáctico con el cual apoyara sus conocimientos en los campos de desarrollo de sitios web enfocándonos en puntos cruciales como son:

Etiquetas Básicas de HTML:

Las etiquetas de código HTML son fragmentos de texto rodeados por paréntesis angulares <>, que tienen funciones y usos específicos y se utilizan para escribir código HTML.

Atributos y Valores:

Los atributos de HTML son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de HTML.
